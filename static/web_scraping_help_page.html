Finding the URL

The URL is the web address of the page you want to scrape. For example, if you want to scrape news from a specific technology section of a website, the URL would be the address of that technology section page.

  1. Open the website in your web browser.
  2. Navigate to the specific page you want to scrape (e.g., the technology news section).
  3. Copy the URL from the address bar of your browser.

For example, to scrape technology news from example.com, you would go to https://www.example.com/technology and copy that URL.

URL: https://www.example.com/technology

Finding the CSS Selector

CSS Selectors are patterns used to select HTML elements on a web page. They help you pinpoint the exact elements you want to extract data from.

  1. Open the website in your web browser.
  2. Right-click on the element you want to select (e.g., a news article title) and choose "Inspect" or "Inspect Element". This will open the browser's developer tools.
  3. In the developer tools, the HTML code of the page will be displayed. The element you right-clicked on will be highlighted.
  4. Right-click on the highlighted HTML element in the developer tools and choose "Copy" > "Copy Selector".

The copied selector is what you'll use in your web scraping tool. For example, to select the title of the first article in a list, the CSS Selector might look like this:

CSS Selector: .article-list .article:nth-child(1) .article-title

Here's how this selector works:

  • .article-list selects an HTML element with the class "article-list"
  • .article:nth-child(1) selects the first element with the class "article" inside the ".article-list" element
  • .article-title selects the element with the class "article-title" inside the selected article.

Finding the XPath

XPath is another language for finding elements in an HTML document. It's more powerful than CSS selectors in some cases, especially for navigating complex HTML structures.

  1. Open the website in your web browser.
  2. Right-click on the element you want to select and choose "Inspect" or "Inspect Element" to open the developer tools.
  3. In the developer tools, find the HTML element you're interested in.
  4. Right-click on the HTML element and choose "Copy" > "Copy XPath". Some browsers may have slightly different wording, but the option will be similar.

Here's an example of an XPath that selects the same element as the CSS Selector example above:

XPath: //*[@id="main-content"]//h2[1]/a

Here's how this XPath works:

  • // selects the current node
  • [@id="main-content"] selects the element with the id "main-content"
  • //h2[1] selects the first h2 element within the selected element
  • /a selects the 'a' tag within the first h2 element.

Finding Element Attributes

Element attributes provide additional information about HTML elements. You can use them to further specify which element you want to select.

  1. Open the website in your web browser.
  2. Right-click on the element you want to select and choose "Inspect" or "Inspect Element". This will open the browser's developer tools.
  3. In the developer tools, look at the HTML code for the element. Attributes are specified within the opening tag of the element.
  4. Right-click on the highlighted HTML element in the developer tools and choose "Copy". The exact wording might vary slightly between browsers (e.g., "Copy Element").

For example, if you inspect the title of a news article, you might see the following HTML:

<h2 class="article-title" id="article-123">News Article Title</h2>

  • class with the value "article-title"
  • id with the value "article-123"

You can use these attributes in your web scraping tool to target this specific element. For example, you could use the ID "article-123" to select this title.
