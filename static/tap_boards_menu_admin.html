Welcome to Boards Menu Admin

Select a menu item from the tree on the left to start editing, or add a new item.


Tree Navigation:

  • Use the chevron to collapse or expand parent items
  • Click a row to edit it in the panel on the right
  • Use Add Root Menu Item and Add Child Item to build the hierarchy

Drag & Drop:

  • Drag an item row to reorder it
  • Drag onto another item to reorder (orange “↕️ Reorder”)
  • Hold Shift while dropping to make it a child (green “➕ Add Child”)
  • Drop on the top root zone to move item to the top of root menus
  • Drop on the bottom root zone to move item to the end of root menus
