# Video Tutorial Files Needed

This directory should contain the following video files for the comprehensive admin pages guide:

## Required Video Files:

1. **interface-overview.mp4**
   - Overview of the admin interface and main navigation
   - Duration: 2-3 minutes recommended

2. **create-page-tutorial.mp4**
   - Step-by-step guide for creating new communication pages
   - Should cover: naming, page creation, and initial setup
   - Duration: 3-4 minutes recommended

3. **button-design-tutorial.mp4**
   - Complete button design and configuration walkthrough
   - Should cover: static vs dynamic buttons, properties, colors, icons
   - Duration: 4-5 minutes recommended

4. **navigation-setup.mp4**
   - How to set up page navigation and linking
   - Duration: 2-3 minutes recommended

5. **organization-strategies.mp4**
   - Best practices for organizing pages and content
   - Duration: 3-4 minutes recommended

6. **advanced-features.mp4**
   - Advanced functionality and tips
   - Duration: 3-5 minutes recommended

## Video Format Requirements:

- **Format**: MP4 (H.264 codec)
- **Resolution**: 1080p preferred, 720p minimum
- **File Size**: Keep under 50MB per video for web performance
- **Audio**: Clear narration with good audio quality

## Usage:

These videos are automatically loaded by the comprehensive admin guide when users click the video tutorial buttons. If a video file is missing, a placeholder message will be shown instead.
